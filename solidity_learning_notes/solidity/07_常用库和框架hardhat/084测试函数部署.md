# 测试环境读取、部署、测试合约

之前部署的时候是javascript环境，现在使用typpescript环境

将hardhat.config.js文件名称修改为hardhat.config.ts

然后添加一个js库
yarn add -D type-node typescript

然后再配置文件中修改为

```javascript
require("@nomicfoundation/hardhat-toolbox");
require("@nomicfoundation/hardhat-ethers");

type Config = import('hardhat/config').HardhatUserConfig

const config: Config = {
  solidity: "0.8.28",
};

export default config;
```

ts文件夹下，编写测试代码
文件名称为合约名称

## 安装测试类型依赖

---

```bash
yarn add -D @types/mocha
```

编写测试代码

运行测试代码，批量运行测试代码

```bash
npx hardhat test
```
