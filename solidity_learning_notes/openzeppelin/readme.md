# OpenZeppelin 专题

本目录用于集中管理 OpenZeppelin 学习内容，按“笔记 + 示例 + 资源”组织。

## 目录说明

- `notes/`：OpenZeppelin 相关理论与模块笔记  
- `examples/`：可运行示例代码（建议每个示例目录带一个 `README.md`）  
- `assets/`：配图与图示资源
